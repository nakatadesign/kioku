# kioku

> Personal knowledge layer for Claude Code.
> Capture → Organize → Retrieve. Proactively.

## What is kioku?

**kioku (記憶)** is a Git-based personal knowledge repository template.
It consolidates three input channels into a single source of truth and proactively pushes relevant knowledge via Telegram notifications.

## Input Channels

| Channel | Method | Session |
|---|---|---|
| Mac Terminal | Claude Code direct | Main session |
| iPhone | Telegram → Claude Code Channels | Same main session |
| Claude Desktop | GitHub Connector | Separate session, same repo |
